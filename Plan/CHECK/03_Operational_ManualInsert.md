## Ruang Lingkup

Modul Manual Insert mencakup input data penjualan/transaksi secara manual oleh operator:
1. **Lihat Data Manual** — Tampil daftar entri manual yang sudah dibuat
2. **Tambah Entri** — Input data penjualan secara manual
3. **Edit / Hapus** — Ubah atau hapus entri yang salah
